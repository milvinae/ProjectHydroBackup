# Hydro Raindrop Whitepaper
Raindrop is a pair of authentication products. Server-side Raindrop is an enterprise-level security protocol to secure APIs and other shared resources, while Client-side Raindrop is a next-gen multi-factor authentication (MFA or 2FA) solution for consumers. [The whitepaper](./Raindrop.md) explains both of these products in greater detail.
